Parameters
deviceName of the serial port. Windows example: "COM1". Linux example: "/dev/ttyS0".
baud_rateBaudrate.
data_bitsNumber of data bits.
stop_bitsNumber of stop bits. Set this parameter to 3 means 1.5 stop bits.
parityParity. It can be even 'e', odd 'o' or nothing 'n'.
flow_controlFlow control. It can be hardware 'h', software 's' or nothing 'n'.
timeoutTimeout in milliseconds for Read and Write.
